# 标准图集类cube 模板，4图

## 介绍
标准图集类模板，适合图集类场景。

## 使用
```
cubetool init <cubeid> -t https://github.com/ntt360/cube-tpl-image-4pic.git
```
或简写：
```
cubetool init <cubeid> -t ntt360/cube-tpl-image-4pic
```
